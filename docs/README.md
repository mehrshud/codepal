Get started with CodePal in just a few minutes. Sign up for an account, upload your code, and receive personalized feedback.

## Installation
To install CodePal, follow these steps:
1. Clone the repository: `git clone https://github.com/mehrshud/CodePal.git`
2. Install dependencies: `npm install`
3. Start the application: `npm start`

## Usage Examples
* Upload your code: `curl -X POST -F "code=@/path/to/your/code" http://localhost:8080/upload`
* Receive feedback: `curl -X GET http://localhost:8080/feedback`
