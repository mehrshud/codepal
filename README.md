## Quick Start
Get started with CodePal in minutes. Simply install, configure, and start coding.

## Installation
To install CodePal, run the following command:
npm install codepal
Or, use our Docker image:
docker pull omnilertlab/codepal

## Usage Examples
### Basic Usage
const codepal = require('codepal');
const analysis = codepal.analyze('console.log("Hello World!");');
console.log(analysis);
### Advanced Usage
const codepal = require('codepal');
const feedback = codepal.feedback('console.log("Hello World!");', { style: 'javascript' });
console.log(feedback);
